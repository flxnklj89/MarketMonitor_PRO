MARKTRISIKO-KOMPASS PRO – DIREKT-UPLOAD FÜR GITHUB

Diese ZIP ist bereits in der korrekten Ordnerstruktur angelegt.
Du kannst den Inhalt 1:1 in dein GitHub-Repository hochladen.

ZIELSTRUKTUR

repo-root/
├─ index.html
├─ documentation.html
├─ manifest.json
├─ data/
│  ├─ latest.json
│  └─ pro_schema.json
├─ scripts/
│  └─ fetch_data.py
└─ .github/
   └─ workflows/
      └─ update-market-data.yml

WICHTIGE HINWEISE

1. index.html ist bereits auf den Marktrisiko-Kompass PRO abgestimmt.
2. documentation.html ist die PRO-Hilfe-/Onboarding-Seite.
3. scripts/fetch_data.py schreibt korrekt nach data/latest.json.
4. .github/workflows/update-market-data.yml ruft korrekt scripts/fetch_data.py auf.
5. data/latest.json ist als sofort nutzbare PRO-Beispieldatei enthalten.

GITHUB SECRET

Unter Settings → Secrets and variables → Actions anlegen:
FRED_API_KEY

EMPFOHLENE REIHENFOLGE

1. Alle Dateien und Ordner aus dieser ZIP in dein Repo hochladen.
2. Prüfen, dass GitHub Pages auf den Repo-Root zeigt.
3. FRED_API_KEY setzen.
4. GitHub Action manuell starten oder auf den nächsten Lauf warten.
